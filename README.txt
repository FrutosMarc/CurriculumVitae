# Création de mon CV en HTML5 et CSS3

Ce CV sera créé sur plusieurs étapes permettant de visualiser les modifications par étape :

- La création d'une structure simple en HTML

- Ajout d'une petite photo

- Finalement suppression de la petite photo pour en mettre une moyenne et ajout d'un formatage CSS3 (Police et présentation). Petite modification du HTML pour l'adaptation CSS3.

- Pour finir je finiole avec du HTML5 (responsive) et des fonds CSS plutôt que des images. Et ajout d'une image pour les titres.
